p5.js Features

Category Feature Description
Core createCanvas() Creates a canvas element in the document.
Core noLoop() Stops draw() from continuously executing.
Core loop() Resumes draw() to continuously execute.
Drawing background() Sets the background color of the canvas.
Drawing fill() Sets the color used to fill shapes.
Drawing stroke() Sets the color used for the outline of shapes.
Drawing ellipse() Draws an ellipse (oval) to the screen.
Drawing rect() Draws a rectangle to the screen.
Input mouseX Returns the current horizontal position of the mouse.
Input mouseY Returns the current vertical position of the mouse.
Input keyPressed() Called once every time a key is pressed.
Transform translate() Moves the origin of the canvas to a different location.
Transform rotate() Rotates the canvas around the origin.
Transform scale() Increases or decreases the size of shapes drawn to the canvas.
Math random() Generates random numbers.
Math map() Re-maps a number from one range to another.
Color color() Creates colors for storing in variables.
Color lerpColor() Blends two colors to find a third color somewhere between them.
